Para poder comenzar a programar en HTML primero devemos saber 2 cosas fundamentales

  • QUE ES HTML?
  • QUE SON LAS ETIQUETAS?

Te dire rapidamente que es cada una de las cosas y ya comenzamos :D

  • HTML (HyperText Markup Language) es el lenguaje de marcado estándar utilizado para crear y estructurar contenido en la web. Consiste en una serie de etiquetas que definen la estructura de una página web, como encabezados, párrafos, imágenes, enlaces, listas, tablas, formularios, entre otros. Cada página web se basa en un documento HTML que le dice al navegador cómo mostrar el contenido. HTML es esencialmente la base sobre la que se construyen las páginas web, y es complementado por otros lenguajes como CSS (para el estilo visual) y JavaScript (para interactividad).
  • Las etiquetas en HTML son los elementos fundamentales que se utilizan para estructurar y organizar el contenido de una página web. Cada etiqueta tiene una función específica y generalmente viene en pares: una etiqueta de apertura y una etiqueta de cierre. Estas etiquetas encierran el contenido al que se les aplica. Una etiqueta se escribe entre signos de menor que (<) y mayor que (>). La etiqueta de apertura inicia el bloque de contenido, y la etiqueta de cierre se escribe con una barra diagonal (/) antes del nombre de la etiqueta para indicar el fin de ese bloque.
